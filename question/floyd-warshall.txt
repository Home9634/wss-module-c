Given a weighted directed graph, find the shortest paths between all pairs of nodes using the Floyd-Warshall algorithm.

Example:
Given a graph with edges [(0, 1, 3), (0, 2, 8), (1, 2, -4), (2, 3, 7)], return the shortest paths from each node to all other nodes.
